# BookManager

BookManager is an Android application that allows users to explore fiction books and search for their favorite titles using the Open Library API.

## Features

- **Browse Fiction**: curated list of fiction books.
- **Search**: Find books by title or author.
- **Book Details**: View detailed information including description, author, and publication date.
- **Favorites**: Save books to your favorites list (stored locally).
- **Pagination**: Infinite scrolling for seamless browsing.
- **Dark Mode**: Toggle between light and dark themes.

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: MVVM (Model-View-ViewModel)
- **Networking**: Retrofit, Moshi
- **Async**: Coroutines, Flow
- **Dependency Injection**: Manual (AppModule)

## Setup

1. Clone the repository.
2. Open in Android Studio.
3. Sync Gradle.
4. Run on Emulator or Device.

## API

Powered by [Open Library API](https://openlibrary.org/developers/api).
